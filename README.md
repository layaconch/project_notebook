# DevOps Operations Platform

轻量化的 DevOps/运维平台模块，提供以下功能：

- Notebook：模仿 Jupyter Notebook 的交互式单元，支持 Markdown/Python/SQL 执行结果查看，并可配置 PostgreSQL/Oracle/MSSQL/CSV 数据源。
- 数据源管理：集中维护各类数据库/文件连接，并提供连通性测试。
- 需求管理：记录并跟踪用户需求、优先级、负责人与状态。
- 问题跟踪：对需求拆分出的 issue 进行登记、处理、关闭。

安装后在应用列表中搜索 “DevOps Operations Platform”，启用后即可在菜单中看到 Notebook、Requirements、Issues 以及 Data Sources 菜单。
